Windows下四元搭建博客网站系统

突发奇想，搭建个博客网站系统吧。网上找了一些方案，折腾了一下午终于搞定，共花费四元用于购买域名，搭建方法待我细细道来。

框架

  • 域名：购于阿里云
  • 文件服务：github
  • 博客系统：hexo
  • 下载安装软件: NodeJs, Git

安装教程

  • 此版本是Mac版本的，一些命令在windows下，需要修正，详细步骤

Windows安装下的坑

  • hexo安装，不要-g，正确命令

  • windows下解决npm, hexo command not found问题

    • 系统变量Path下，新增npm和hexo目录；注意更改成自己的目录名
